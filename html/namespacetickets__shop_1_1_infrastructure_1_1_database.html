class tickets_shop.Infrastructure.Database.AppDbContext ( DbContextOptions< AppDbContext > options)

The primary DbContext for the application, representing the session with the database. It maps domain entities to database tables and configures entity relationships.

Parameters
optionsThe options required to configure the context.

Represents the collection of all User entities (including Admins and RegularUsers).

Represents the collection of Ticket entities.

Represents the collection of Event entities.

A factory method to create an instance of AppDbContext configured to use SQLite.

Returns
A new instance of AppDbContext.

Configures the User hierarchy to use just one table and distinguish roles based on the string 'Discriminator' column.

Configures the one-to-many relationship between RegularUser and Ticket (OwnedTickets). Sets a cascade delete behavior (if the user gets deleted so do the tickets connected to that user).

Configures the one-to-many relationship between Admin and Event (OrganizedEvents). Sets a cascade delete behavior (if the admin gets deleted so do the events connected to that admin).

Configures the one-to-many relationship between Event and Ticket (Tickets). Sets a cascade delete behavior.

Overrides the base method to apply all custom entity and relationship configurations.
