enum tickets_shop.Application.ServiceHandlers.PurchaseResult

Represents the possible outcomes of a ticket purchase attempt.

Enumerator
Success 

The purchase was completed successfully.

NoTicketsAvailable 

The event does not have the requested number of tickets available.

NotEnoughMoney 

The user does not have enough money to complete the purchase.
