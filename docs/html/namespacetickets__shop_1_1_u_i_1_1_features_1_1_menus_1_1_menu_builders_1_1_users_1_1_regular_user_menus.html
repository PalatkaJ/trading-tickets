class tickets_shop.UI.Features.Menus.MenuBuilders.Users.RegularUserMenus.RegularUserBrowseEventsMenuBuilder ( ApplicationState applicationState)

The concrete menu builder for the Events Browser screen for Regular Users. It displays a list of all events that currently have tickets available for sale.

Parameters
applicationStateThe application's shared state container.

Attempts to create a selectable MenuItem for a given Event if tickets are available. If created, the action sets the current event and navigates to the event submenu.

Parameters
eThe Event entity to potentially list.
itemsThe list to which the MenuItem is added.
Returns
True if the item was created (tickets are available); otherwise, false.

Retrieves all Event entities from the repository using a lazy-loading IQueryable pattern.

Returns
An IQueryable collection of all events with necessary dependencies.

Builds the menu by iterating through all available events and listing those with tickets.

Parameters
itemsThe list to which custom MenuItems are added.
